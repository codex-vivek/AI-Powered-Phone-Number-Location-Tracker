# 📡 AI-Powered Phone Number Location Tracker
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/deploy?repository=https://github.com/codex-vivek/AI-Powered-Phone-Number-Location-Tracker&branch=main&main_module_filename=app.py)

A high-performance, autonomous mobile tracking system with police-level tactical interface. This system simulates real-time location triangulation and SS7 protocol intercepts.

## 🚀 Features
- **SS7 Signal Intercept**: Real-time packet capture simulation from MSC nodes.
- **Autonomous Satellite Mapping**: High-precision Google Hybrid Intelligence view.
- **HLR Cache Recovery**: Automatically pulls last known location if the device is powered off.
- **Device Forensics**: Live battery status, signal strength, and LAC/Cell-ID tracking.
- **Intelligence Reports**: AI-driven safety and threat analysis of the target device.

## 🛠️ Technology Stack
- **Frontend**: Streamlit
- **Mapping**: Folium / Streamlit-Folium
- **Geocoding**: GeoPy, OpenCage, Nominatim
- **Logic**: Phonenumbers, Pandas, Scikit-learn

## 📦 Installation
1. Clone the repository:
```bash
git clone https://github.com/codex-vivek/AI-Powered-Phone-Number-Location-Tracker.git
cd AI-Powered-Phone-Number-Location-Tracker
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run the app:
```bash
streamlit run app.py
```
